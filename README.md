# 🎮 Kelime Oyunlarım

Bu repo, Python öğrenme sürecimde geliştirdiğim kelime tabanlı oyunları içerir.

## 📁 Projeler

### 🕹️ Adam Asmaca (CLI)

* Terminal tabanlı versiyon
* Rastgele kelime seçimi
* Harf tahmin sistemi
* Can mekanizması

### 🖥️ Adam Asmaca (GUI)

* Tkinter ile geliştirilmiş arayüz
* Kullanıcıdan input alma (Entry)
* Buton ile tahmin etme
* Dinamik ekran güncelleme

### 🔤 Kelime Oyunu

* Temel Python mantığı ile geliştirilmiş
* String ve liste işlemleri kullanıldı

---

## 🧠 Kullanılan Teknolojiler

* Python 🐍
* Tkinter (GUI için)
